# bletemps

The embedded half of the `bletemps` codebase. Built for ESP32 using the
[ESP-IDF Framework](https://docs.espressif.com/projects/esp-idf/en/stable/esp32/api-reference/index.html).
Currently configured for deployment on an [ESP32-S3 Super Mini](https://www.espboards.dev/esp32/esp32-s3-super-mini/)
